# Personal Profile Card

A modern and responsive personal profile card built with **HTML**, **CSS**, and **JavaScript**.

This project was created as part of my web development learning journey to strengthen my understanding of frontend fundamentals while building something visually appealing and interactive.

---

## ✨ Features

* 🌗 Light & Dark mode toggle
* 💾 Theme preference saved using `localStorage`
* 🎨 Modern and responsive UI
* ✨ Neon hover effect in Dark Mode
* 👤 Circular profile picture
* 🏷️ Skill badges
* 🔗 GitHub and LinkedIn buttons with icons
* 📱 Mobile-friendly layout

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla)
* Font Awesome
